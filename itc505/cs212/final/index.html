Revenge of Lights Out

Addendum: Development Process and Insights

Process:

  1. Understanding the Requirements: The initial step was to understand the requirements for the Revenge of Lights puzzle game.
  2. Creating HTML Structure: A basic HTML structure was created with a title, game board container, and individual squares.
  3. Styling with CSS: CSS styles were designed for a visually appealing layout, including flexbox for centering elements.
  4. JavaScript Logic: The JavaScript code handled game logic, such as creating a solvable board, toggling square colors, and checking for a win condition.
  5. Testing and Debugging: Thorough testing in various browsers and using developer tools for debugging were crucial steps.
  6. User Interaction: An alert was added to notify the user when they win the game, enhancing the overall user experience.

Creative Choices:

  • Styling: A clean and simple design was chosen with a contrasting background color for visual appeal.
  • Font Family: A standard sans-serif font was selected for readability and a modern look.
  • Flexbox: Flexbox was used for centering elements, providing a responsive design.

Challenges Faced and Solutions:

  1. Creating a Solvable Board: Ensuring a solvable board was challenging; the logic was modified to toggle every other square during the initial setup.
  2. User Interface: Improving the user interface involved adding a title and adjusting styling for better visibility.

Insights Gained:

  • Understanding Game Logic: Insights were gained into creating solvable puzzles and handling user interactions in a game-like scenario.
  • CSS Styling for Readability: The importance of CSS styling for readability and user engagement was emphasized.
